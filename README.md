# DevJourney API

API  desenvolvida em Python com FastAPI para registrar projetos de portfólio e acompanhar a evolução como desenvolvedor.

## Tecnologias utilizadas

- Python
- FastAPI
- Uvicorn
- Pydantic

## Funcionalidades

- Cadastrar projetos
- Listar projetos
- Buscar projeto por ID
- Remover projeto
- Gerar resumo da jornada dev
- Documentação automática com Swagger
